# ServiceForce develop tool

基于 electron + vite + vue3 + pnpm + typescript 构建的桌面应用，助力前端开发

---

![run-capture](./docs/assets/capture.png)

---

## 安装依赖

```
pnpm i
```

## 开发

```
pnpm run dev
```

## 构建

```
pnpm run build
```

## 上传可执行文件

```
pnpm run upload
```

## 文档开发

```
pnpm run docs:dev
```
